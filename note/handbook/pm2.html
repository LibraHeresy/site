PM2 官方手册

命令 ​

pm2 start（启动） ​

shell
# 启动应用
pm2 start app.js

# 启动应用并监听文件改动
pm2 start app.js --watch

pm2 restart（重启） ​

shell
# 重启指定应用
pm2 restart app

# 重启所有应用
pm2 restart all

# 重启多个应用
pm2 restart app1 app2 app3

pm2 reload（重载） ​

shell
# 重载指定应用
pm2 reload app

# 重载所有应用
pm2 reload all

# 重载多个应用
pm2 reload app1 app2 app3

pm2 stop（停止） ​

shell
# 停止指定应用
pm2 stop app

# 停止所有应用
pm2 stop all

# 停止多个应用
pm2 stop app1 app2 app3

pm2 delete（删除） ​

shell
# 删除指定应用
pm2 delete app

# 删除所有应用
pm2 delete all

# 删除多个应用
pm2 delete app1 app2 app3

pm2 startup（开启自启） ​

shell
# 使用进程列表配置，开启开机自启动
pm2 startup

pm2 save（保存进程列表） ​

shell
# 保存进程列表用于开机自启配置
pm2 save

pm2 unstartup（禁用进程列表配置） ​

shell
# 禁用进程列表
pm2 unstartup

pm2 cleardump（清空进程列表配置） ​

shell
# 新建一个空的配置文件覆盖
pm2 unstartup

pm2 update（更新） ​

shell
# 更新进程
pm2 update

pm2 list（显示） ​

shell
# 显示所有进程状态
pm2 list

pm2 link（链接官方在线监控） ​

  1. 访问 pm2.io

  2. 执行 pm2 link XXX XXX，关联服务

  3. 执行 pm2 plus，自动打开在线监控网页
